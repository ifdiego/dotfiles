# dotfiles

All my dotfiles managed with [GNU Stow](https://www.gnu.org/software/stow/). If you're interested in stow, I recommend [this video](https://www.youtube.com/watch?v=y6XCebnB9gs) for a quick introduction on why you might want to use it.

First, clone this repository into `~/dotfiles`.

```bash
git clone https://github.com/ifdiego/dotfiles.git ~/dotfiles
cd ~/dotfiles
stow .
```
